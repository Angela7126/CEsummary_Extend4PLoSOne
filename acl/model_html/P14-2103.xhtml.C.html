[0] This paper introduces an unsupervised graph-based method that selects textual labels for automatically generated topics. [1] Our approach uses the topic keywords to query a search engine and generate a graph from the words contained in the results. [2] PageRank is then used to weigh the words in the graph and score the candidate labels. [3] The state-of-the-art method for this task is supervised [ 14 ]. [4] Evaluation on a standard data set shows that the performance of our approach is consistently superior to previously reported methods. [5] We described an unsupervised graph-based method to associate textual labels with automatically generated topics. [6] Our approach uses results retrieved from a search engine using the topic keywords as a query. [7] A graph is generated from the words contained in the search results metadata and candidate labels ranked using the PageRank algorithm. [8] Evaluation on a standard data set shows that our method consistently outperforms the supervised state-of-the-art method for the task.
