[1] We have described a new (publicly available) corpus for the task of verbal irony detection.. [2] The data comprises comments scraped from the social news website reddit.. [3] We recorded confidence judgements and requests for contextualizing information for each comment during annotation.. [4] We analyzed this corpus to provide empirical evidence that annotators quite often require context beyond the comment under consideration to discern irony; especially for those comments ultimately deemed as being intended ironically.. [5] We demonstrated that a standard token-based machine learning approach misclassified many of the same comments for which annotators tend to request context.. [6] We have shown that annotators rely on contextual cues (in addition to word and grammatical features) to discern irony and argued that this implies computers should, too.. [7] The obvious next step is to develop new machine learning models that exploit the contextual information available in the corpus we have curated (e.g.,, previous comments by the same user, the thread topic..
