[1] Robust citation classification has been hindered by the relative lack of annotated data.. [2] In this paper we successfully use a large, out-of-domain, annotated corpus to improve the citation polarity classification.. [3] Our approach uses a deep learning neural network for domain adaptation with labeled out-of-domain data and unlabeled in-domain data.. [4] This semi-supervised domain adaptation approach outperforms the in-domain citation polarity classification and other fully supervised domain adaptation approaches.. [5] Acknowledgments..
