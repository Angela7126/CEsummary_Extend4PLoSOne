[1] We have shown in this work how temporal information in the form of timelines can be incorporated into multi-document summarization.. [2] We achieve this through two means, using.. [4] To overcome errors propagated from the underlying temporal processing systems, we proposed a reliability filtering metric which can be used to help decide when temporal information should be used for summarization.. [5] The use of this metric leads to an overall 5.9% gain in R-2 over the competitive SWING baseline..
