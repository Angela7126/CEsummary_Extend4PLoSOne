[1] In this paper, we have shown that while using parse accuracy in a simple reranking strategy for self-monitoring fails to improve BLEU scores over a state-of-the-art averaged perceptron realization ranking model, it is possible to significantly increase BLEU scores using an SVM ranker that combines the realizer s model score together with features from multiple parsers, including ones designed to make the ranker more robust to parsing mistakes that human readers would be unlikely to make.. [2] Additionally, via a targeted manual analysis, we showed that the SVM reranker frequently manages to avoid egregious errors involving vicious ambiguities, of the kind that would mislead human readers as to the intended meaning.. [3] As noted in Reiter s [ 30 ] survey, many NLG systems use surface realizers as off-the-shelf components.. [4] In this paper, we have focused on broad coverage surface realization using widely-available PTB data where there are many sentences of varying complexity with gold-standard annotations following the common assumption that experiments with broad coverage realization are (or eventually will be) relevant for NLG applications..
