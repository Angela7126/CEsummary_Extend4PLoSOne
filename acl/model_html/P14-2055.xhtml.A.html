[1] In order to summarize a document, it is often useful to have a background set of documents from the domain to serve as a reference for determining new and important information in the input document.. [2] We present a model based on Bayesian surprise which provides an intuitive way to identify surprising information from a summarization input with respect to a background corpus.. [3] Specifically, the method quantifies the degree to which pieces of information in the input change one s beliefs about the world represented in the background.. [4] We develop systems for generic and update summarization based on this idea.. [5] Our method provides competitive content selection performance with particular advantages in the update task where systems are given a small and topical background corpus..
