[0] This paper introduces a new technique for phrase-structure parser analysis, categorizing possible treebank structures by integrating regular expressions into derivation trees. [1] We analyze the performance of the Berkeley parser on OntoNotes WSJ and the English Web Treebank. [2] This provides some insight into the evalb scores, and the problem of domain adaptation with the web data. [3] We also analyze a test-on-train dataset, showing a wide variance in how the parser is generalizing from different structures in the training material.
