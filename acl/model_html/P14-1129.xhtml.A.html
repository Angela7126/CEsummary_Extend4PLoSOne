[1] Recent work has shown success in using neural network language models (NNLMs) as features in MT systems.. [2] Here, we present a novel formulation for a neural network joint model (NNJM), which augments the NNLM with a source context window.. [3] Our model is purely lexicalized and can be integrated into any MT decoder.. [4] We also present several variations of the NNJM which provide significant additive improvements.. [5] Although the model is quite simple, it yields strong empirical results.. [6] On the NIST OpenMT12 Arabic-English condition, the NNJM features produce a gain of +3.0 BLEU on top of a powerful, feature-rich baseline which already includes a target-only NNLM.. [7] The NNJM features also produce a gain of +6.3 BLEU on top of a simpler baseline equivalent to Chiang s [ 5 ] original Hiero implementation.. [8] Additionally, we describe two novel techniques for overcoming the historically high cost of using NNLM-style models in MT decoding.. [9] These techniques speed up NNJM computation by a factor of 10,000x, making the model as fast as a standard back-off LM..
