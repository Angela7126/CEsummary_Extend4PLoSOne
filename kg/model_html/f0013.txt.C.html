[0] The Semantic Link Network (SLN) is a loosely coupled semantic data model for managing Web resources. [1] Its nodes can be any type of resource. [2] Its edges can be any semantic relation. [3] Potential semantic links can be derived out according to reasoning rules on semantic relations. [4] This paper proposes the schema theory for the SLN, including the concepts, rule-constraint normal forms, and relevant algorithms. [5] The theory provides the basis for normalized management of semantic link network. [6] A case study demonstrates the proposed theory. [7] The SLN schema regulates the semantics of semantic link networks so as to manage semantic link network instances. [8] The proposed rule-constraint normal forms can help manage and maintain the SLN schema efficiently. [9] Two algorithms for SLN schema extension and reduction are introduced. [10] Reasoning algorithms for deriving more semantic relations have been proposed. [11] The proposed SLN schema and relevant theory are important parts of the SLN model.
