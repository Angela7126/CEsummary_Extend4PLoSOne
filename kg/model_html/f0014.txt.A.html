[0] Well-defined intellectual problems cam be regarded as those of inverting functions and partial functions defined by Turing machines. [1] If we are given a Turing machine and the output of it, is there anyway to find out the input? [2] That is, can we find the inversion function defined by this Turing machine? [3] This work intended to discuss such a question. [4] First, we discuss the existence of inversion function on some conditions. [5] Then we tried to improve the purely enumerative method's efficiency taking into account some of the structure of Turing machines. [6] In order to discuss this problem, we also attempted to precisely define the meaning of efficiency. [7] Finally we listed some place worthy of remark in the last part.
