[0] Database systems have not traditionally supported science's core data type: the N-dimensional array. [1] This collection of problems is generally called the impedance mismatch -- meaning the mismatch between the programming model and the database capabilities. [2] They are being integrated with programming languages so that they can support object-oriented databases. [3] This new generation of object relational database systems treats any data type as an encapsulated type that can be stored as a value in a field of a record. [4] This linking-embedding object model nicely accommodates the integration of database systems and file systems -- files are treated as linked-objects.
