[0] file systems have no metadata beyond a hierarchical directory structure and file names. [1] They encourage a do-it-yourself- data-model that will not benefit from the growing suite of data analysis tools. [2] They encourage do-it-yourself-access-methods that will not do parallel, associative, temporal, or spatial search. [3] They also lack a high-level query language. [4] We see NetCDF, HDF, FITS, and Google Map-Reduce as nascent database systems (others might think of them as file systems).
