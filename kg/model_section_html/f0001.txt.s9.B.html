[1] This indicates that human mind uniformly processes various things at certain cognitive level. [2] This indicates the possibility of creating a unified method for processing different objects such as texts, pictures, videos and graphs. [3] A unified representation should reflect the most fundamental characteristics of concerned representations. [4] Transformation between representations is a way to realize unification. [5] Establishing a unified representation enables a summarization system to uniformly process various representations. [9] The cognitive hierarchy of different communities corresponds to different sub-graphs of the hierarchy and has different basic cognitive levels. [10] A summary is suitable if its cognitive level is the same as the reader's cognitive level. [11] (1) If the cognitive level of the original representation is the same as the reader's cognitive level, the summary should use the core representations in the original representation. [12] (2) If the cognitive level of the original representation is higher than the reader's cognitive level, the summary should use more specific concepts in the commonsense category hierarchy. [13] (3) If the cognitive level of the original representation is lower than the reader's cognitive level, the summary should use more general concepts in the commonsense category hierarchy.
