[0] As mentioned earlier, scientists often start with numeric data arrays from their instruments or simulations. [1] Often, these arrays are accompanied by tabular data describing the experimental setup, simulation parameters, or environmental conditions. [2] the arrays undergo feature extraction to produce objects that are the basis for further analysis. [3] Most scientific studies involve exploring and data mining these object-oriented tabular datasets. [4] Performing this filter-then-analyze, data analysis on large datasets with conventional procedural tools runs slower and slower as data volumes increase. [5] the science community must create a synthesis of database systems and file systems. [6] Set-oriented file processing will make file names increasingly irrelevant -- analysis will be applied to "all data with these attributes" rather than working on a list of file/directory names or name patterns.
